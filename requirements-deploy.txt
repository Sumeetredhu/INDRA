# top level), so the API boots and every endpoint works — the omitted packages only unlock extra
# backends and modalities:
#
#   chromadb              -> in-process numpy vector store is used instead (D1)
#   neo4j, redis          -> in-process graph / event bus / cache are used instead (D1)
#   google-generativeai   -> the Gemini provider talks to the REST API over httpx anyway (D2)
#   groq                  -> same, httpx
#   python-magic          -> validation.py implements its own magic-number table; the Linux wheel
#                            also needs a libmagic system library that free tiers do not ship
#   whisper/gtts/spacy/ultralytics/easyocr -> optional modalities, degrade with a logged warning
#
# Use requirements.txt for a full local or Docker install.
